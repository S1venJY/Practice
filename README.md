# Practice

## Опис проєкту
Practice — навчальний репозиторій для виконання практичних завдань з програмування.

## Мета проєкту
Метою проєкту є закріплення практичних навичок програмування та роботи з GitHub.

## Використані технології
- Мова програмування: C#
- Платформа: .NET
- Git, GitHub

## Встановлення та запуск
Проєкт використовується у навчальних цілях.
Для роботи необхідно клонувати репозиторій:

git clone https://github.com/S1venJY/Practice.git

## Примітка
Проєкт створений у межах навчальної практики.
